### idea配置多环境启动

#### 1 pom添加以下内容
在pom文件<project>标签下添加如下内容
````xml
<!--配置环境的profile-->
    <profiles>
        <profile>
            <id>dev</id>
            <properties>
                <!--使用${environment}获取值-->
                <environment>dev</environment>
            </properties>
        </profile>
        <profile>
            <id>test</id>
            <properties>
                <environment>test</environment>
            </properties>
        </profile>
        <profile>
            <id>prod</id>
            <properties>
                <environment>prod</environment>
            </properties>
        </profile>
    </profiles>
````
#### 2 创建配置文件
application.yml、application-dev.yml、application-prod.yml、application-test.yml
在application.yml文件当中添加一下配置
````yaml
spring:
  application:
    name: ruoyi-service
  profiles:
    active: dev
````
注意：一次只能选一个环境，不然默认是第一次选定的环境

#### 3 maven打包
命令如下
````shell
mvn clean install -P prod
````

#### 4 jar 启动
````shell
java -jar ruoyi-admin.jar --spring.profiles.active=prod
````
